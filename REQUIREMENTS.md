## Core Mechanics
- Top-down 3D perspective Tower Defense.
- Mazeing mechanic: Players place towers on a grid to create the path for enemies to follow.

## Tech Stack
- Frontend: React with TypeScript.
- Build Tool: Vite.
- 3D Rendering: React Three Fiber (R3F) and Drei.

## Visual Style & Flavor
- Visuals: Low-poly cartoon aesthetic.
- Humorous Elements:
  - Enemies display goofy text/chat bubbles upon death.
  - Bosses display text bubbles upon arrival.
- Narrator: A "Tour Guide Narrator" NPC that dynamically narrates happenings and observes the game state, adding a funny and interactive touch.
